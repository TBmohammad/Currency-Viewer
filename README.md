Initial release of Currency Viewer v1.0

Currency Viewer is a Flask-based market dashboard designed to be lightweight, simple, and suitable for Python hosting environments.

Features:
- Cryptocurrency price tracking
- Forex exchange rates
- Commodities market data
- Fiat currency support
- Dark mode interface
- Responsive design
- PWA support
- Ready for deployment on Python hosting / cPanel

Built with Flask, Requests, and external market APIs.
